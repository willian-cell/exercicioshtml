Sobre

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Com HTML, você pode estruturar conteúdo, criar links, listas e muito mais.

Tag <strong>

A tag <strong> é usada para definir texto com importância forte. O texto envolvido com essa tag é exibido em negrito.

<strong>Texto em negrito</strong>

Exemplo: Texto em negrito

Tag <em>

A tag <em> é usada para definir texto que deve ser enfatizado. O texto envolvido com essa tag é exibido em itálico.

<em>Texto em itálico</em>

Exemplo: Texto em itálico

Tag <mark>

A tag <mark> é usada para marcar texto com uma cor de fundo destacada.

<mark>Texto destacado</mark>

Exemplo: Texto destacado

Tag <abbr>

A tag <abbr> é usada para definir uma abreviação ou acrônimo. Você pode fornecer uma descrição completa com o atributo title.

<abbr title="Hypertext Markup Language">HTML</abbr>

Exemplo: HTML

Tags <ol> e <ul>

A tag <ol> define uma lista ordenada (numerada), enquanto a tag <ul> define uma lista não ordenada (com marcadores).

<ol>
<li>Item 1</li>
<li>Item 2</li>
</ol>
<ul>
<li>Item A</li>
<li>Item B</li>
</ul>

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2

Exemplo de lista não ordenada:

  • Item A
  • Item B

Tag <video>

A tag <video> é usada para incorporar vídeos em uma página web. Ela permite que você adicione controles como play, pause e volume.

<video width="320" height="240" controls>
<source src="video.mp4" type="video/mp4">
Seu navegador não suporta o elemento <video>.
</video>

Tag <iframe>

A tag <iframe> é usada para incorporar outra página web dentro da página atual. É comumente usada para exibir conteúdos de outras fontes, como vídeos do YouTube.

<iframe width="560" height="315" src="https://www.youtube.com/embed/5Hn58p-hYC0" frameborder="0" allowfullscreen></iframe>

Tag <img>

A tag <img> é usada para incorporar imagens em uma página web. Ela não possui uma tag de fechamento e requer os atributos src (caminho da imagem) e alt (texto alternativo).

<img src="imagem.jpg" alt="Descrição da imagem" width="300">

Exemplo:

Tag <figure>

A tag <figure> é usada para encapsular uma imagem (ou outro conteúdo gráfico) e sua legenda. Ela é geralmente usada com a tag <figcaption>.

<figure>
<img src="https://t.ctcdn.com.br/HfaAfdSP0yElfXWQelkeEQvkgyU=/1000x563/smart/i257652.jpeg" alt="Descrição da imagem" width="300">
<figcaption>ARARA AZUL</figcaption>
</figure>
